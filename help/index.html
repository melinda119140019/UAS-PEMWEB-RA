Tentang Website

Selamat datang di Portal Data Mahasiswa Universitas Imajinasi. "Universitas Imajinasi" akan menggunakan situs web ini sebagai "Sistem Pendataan Mahasiswa" untuk menangani data dan informasi mahasiswa. Halaman beranda (indeks) dan halaman formulir adalah dua halaman utama yang tersedia di situs web ini. Halaman tabel adalah fitur lain yang memungkinkan pengguna untuk melihat data mahasiswa yang telah dimasukkan menggunakan formulir.

Website sederhana ini dibuat untuk memenuhi Ujian Tengah Semester Pemrograman Web Tahun Ajaran 2023/2024 Semester Ganjil.

Halaman beranda (index) memiliki tampilan dengan latar belakang berwarna biru dan header dengan judul "Selamat Datang di Sistem Pendataan Mahasiswa". Pada halaman ini terdapat menggunakan menu navigasi di header untuk mengakses informasi tentang website ("Tentang Website") dan informasi tentang saya ("Identitas").

Halaman formulir berisi formulir pengisian data mahasiswa dengan beberapa input yaitu nama, NIM, program studi, email, password, jenis kelamin, tanggal lahir, dan alamat. Ketika sudah selesai mengisi formulir dan menekan tombol "Daftar", data mahasiswa akan disimpan dalam penyimpanan lokal browser (localStorage).

Halaman tabel menampilkan data mahasiswa yang telah diisi melalui formulir. Data-data ini sebagian dibuat secara dummy dan sebagian hasil inputan dari saya yang tersimpan di penyimpanan lokal browser dan ditampilkan dalam bentuk tabel. Kita juga dapat kembali ke halaman beranda atau menuju halaman formulir melalui menu navigasi.
